##### 각 기능에 필요한 Component
  
- 글 목록 보기 기능 (리스트 형태)
    - PostList, PostListItem
  
- 글 보기 기능
    - Post
  
- 댓글보기 기능
    - CommentList, CommentListItem
  
- 글 작성 기능
    - PostWrite
  
- 댓글 작성 기능
    - CommentWrite
  

##### 폴더 구성
            
- list -> 리스트와 관련된 Component들을 모아놓은 폴더
- page -> 페이지 Component들을 모아놓은 폴더
- ui -> UI Component들을 모아놓은 폴더
